Implementasi manajemen data mahasiswa menggunakan struktur data **Singly Linked List** dalam bahasa **Java** dan **Python**.

Program ini dibuat untuk mengelola data mahasiswa (Nama dan NIM) dengan konsep Linked List manual tanpa menggunakan library eksternal. Setiap item (Node) berisi NIM, Nama, dan Pointer ke node berikutnya.

## ğŸš€ Fitur Menu
1. **Insert at Beginning**: Data baru ditempatkan di depan.
2. **Insert at Given Position**: Data baru ditempatkan sesuai pilihan (1 sampai count+1).
3. **Insert at End**: Data baru ditambahkan di akhir.
4. **Delete from Beginning**: Hapus data paling depan.
5. **Delete Given Position**: Hapus data di posisi pilihan (1 sampai count).
6. **Delete from End**: Hapus data terakhir.
7. **Delete First Occurrence**: Hapus data berdasarkan NIM sesuai input user.
8. **Show Data**: Tampilkan semua data di LinkedList.
9. **Exit**: Keluar dari program.
